VERSION MANAGEMENT

WesMC Utils Bundle delivers a Hubitat library and a demonstration file.

  • Hubitat tracks the relationship between instantiated Apps & Devices and their footing in Apps Code, Drivers Code and Libraries Code.
  • Hubitat Bundles can include Apps Code, Drivers Code and/or Libraries Code.
  • Hubitat DOES NOT provide version control of Apps Code, Drivers Code or Libraries Code.

  • The only way for two versions of code (App, Driver or Library) to coexist is to include version information in the filename (i.e., of the App, Driver and Library).
